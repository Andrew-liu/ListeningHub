Features

八大核心功能

从基础听读到精进复习，一站式覆盖语言学习全流程

01

多集节目点播

精选 VOA Special English 多期节目，涵盖健康、科学、习语等主题。一键切换集数，支持整集顺序播放或跳转任意句子。

健康 科学 习语
02

逐句精听

每一句都可独立播放、循环、回放。配合三档语速调节（慢 / 中 / 快），从基础到进阶循序渐进，把每个发音都听懂、听透。

Space 播放 ←→ 切换
03

中英双语对照

每句英文配备精准中文翻译，一键切换显示 / 隐藏。先听后看、边听边看、纯听力模式，灵活适配各阶段练习需求。

示例
Exercise is good for health.
锻炼有益于健康。
04

智能生词本

点击任意英文单词即可弹出音标、发音、中英双解；一键收藏到生词本，自动保存到本地，随时复习。

exercise /ˈeksərsaɪz/
中 体育锻炼；运动
05

听写模式

只听音频写下每一个单词，系统即时对比评分。逐词高亮差异 · 0-100 分评估 · 训练拼写与听辨的绝佳方式。

exercise ✓ 正确　excersise ✗ 拼错
06

跟读模式

麦克风录制你的发音，与原音一键对比回放。模仿语音语调、节奏停顿，打磨真正地道的英语口语。

基于 MediaRecorder API · 浏览器原生录音
07

间隔重复复习 (SRS)

采用简化版 SM-2 算法，根据你对单词的熟练度，智能安排下次复习时间。记得牢、忘得慢。

忘记 困难 还行 熟练
08

个性化体验

三色主题（明亮 / 暗黑 / 复古）· PWA 离线支持，可安装到桌面 · 导入真实 VOA MP3 音频 · Anki / CSV 导出生词。
